## 下载

### Docker

```
docker pull apachecn0/nlp-py-2e-zh
docker run -tid -p <port>:80 apachecn0/nlp-py-2e-zh
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install nlp-py-2e-zh
nlp-py-2e-zh <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g nlp-py-2e-zh
nlp-py-2e-zh <port>
# 访问 http://localhost:{port} 查看文档
```
